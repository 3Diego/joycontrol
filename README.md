# joycontrol
Emulate Nintendo Switch Controllers over Bluetooth.

Work in progress.

Pairing works, emulated controller shows up in the "Change Grip/Order" menu of the Switch.

Tested with Raspberry Pi 4B Raspbian GNU/Linux 10 (buster)
